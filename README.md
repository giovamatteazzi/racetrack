# RaceTrack
progetto per il corso di Progettazione Web.
Linguaggi: HTML, CSS, JS, PHP

---

## Uso
Scaricare XAMPP, salvare il progetto in htdocs/, avviare Apache e MySQL dentro XAMPP e digitare in un browser https://localhost/racetrack.
Il gioco consiste nell'interpretazione geometrica di una gara su un tracciato tra due macchine, con movimenti che simulano una guida fisica, fatta di accelerazioni e frenate. Per maggiori informazioni, consultare l'apposita guida del sito o leggere qui https://it.wikipedia.org/wiki/Formula_1_(gioco).
